#### 1 引用的定义
- **作用**：给变量起别名，使用等同原变量；
- **语法**：```数据类型 &别名 = 变量名```
- **注意事项**：一定要初始化，初始化后不能更改，函数返回引用可以作为左值；
```cpp
int main() {    
    // 引用
    int a = 10;
    int c = 20;
    int &b = a; // 引用必须初始化
    // &b = c; // error, 初始化后不可更改
    cout << "a = " << a << " b = " << b << endl;
    b = 20;
    cout << "a = " << a << " b = " << b << endl;
}
```
stdout:
```
a = 10 b = 10
a = 20 b = 20
```

#### 2 引用作函数参数
- **作用**：函数传参时，可以通过引用让形参修饰实参，简化指针修改实参用法
- **语法**：函数定义```void funcA(int &a, int &b)```, 函数调用```funcA(a, b)```
